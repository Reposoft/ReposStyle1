ReposStyle    

A free open source stylesheet for the Subversion index web page. The XSLT and CSS is fully customizable. There is icons for folders and many different filetypes. Install Repos Style as-is or use as a sample XSLT to design your own. It also includes a simple log viewer.

You need the repository configured in Apache. See the Subversion book. Repos Style is installed with an SVNIndexXSLT directive. To use the log viewer you need PHP.

The advantages of using XSLT instead of a viewer like ViewSVN is that it is lightweight and that you use the real URLs in the directory listing. There is no viewer for older revisions, but you can customize the stylesheet to link to other tools.

From

to

in five minutes

Start by verifying that your http repository is up and running.

  1. Download Repos Style zip
  2. Unzip to the document root of the repository host (this creates a folder /repos)
  3. In the Apache config, set SVNIndexXSLT "/repos/view/repos.xsl"
  4. Reload Apache
  5. Check your http URL again

How to change install path

  • Rename the unzipped folder from /repos to anything
  • Update the SVNIndexXSLT
  • In the XSLT files (repos/log.xls and view/log.xsl if used), change the xsl:param "static" to your new folder name.

How to customize

Check the configuration section in the XSL files.

It is good practice to have a start page on the repository host with links to each project's trunk, branches, tags folder(s). That way you don't need to give read access to repository root (which would be inherited by new projects by default). Repos Style supports a startpage concept, meaning that when you click up from trunk you get to the server startpage. To disable this, empty the startpage parameter in repos.xsl.

To customize the menu, edit the commandbar section of the xsl. To customize the buttons displayed on each row, edit the actions section of the dir or file templates.

See the development section for how to contribute changes.
